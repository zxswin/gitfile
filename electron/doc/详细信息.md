# 详细信息

- 安装 Electron

```bash
# 安装electron依赖
npm install electron --save-dev

# 自定义安装
##  在x64机器上安装ia32位版本
npm install --arch=ia32 electron

## 使用 --platform 来指定开发平台 (例如, win32、linux 等):
npm install --platform=win32 electron

# 安装最新稳定版本
npm install --save-dev electron

# 更新到最新稳定版本
npm install --save-dev electron@latest
```

- 用 asar 打包 App 源代码

```bash
为缓解 Windows 下路径名过长的 问题， 略微加快一下 require的速度以及隐藏你的源代码，你可以选择把你的应用打包成 asar档案文件，这只需要对你的源代码做一些很小的改动。
electron-packager,、electron-forge和electron-builder中都得到了支持，开箱即用
asar 是一种将多个文件合并成一个文件的类 tar 风格的归档格式。 Electron 可以无需解压整个文件，即可从其中读取任意文件内容。

# 生成 asar 档案文件
1. 安装 asar 实用程序
$ npm install -g asar

2. 使用 asar pack 打包
$ asar pack your-app app.asar

# 使用 asar 档案文件
# Node API 和 Chromium 提供的 Web API 都是支持从档案文件中读取文件
# 从 asar 档案读取一个文件：
const fs = require('fs')
fs.readFileSync('/path/to/example.asar/file.txt')

# 在 Web 页面里，用 file: 协议可以获取 asar 包中文件。
# 用 $.get 获取文件:
let $ = require('./jquery.min.js')
$.get('file:///path/to/example.asar/file.txt', (data) => {
  console.log(data)
})
</script>


```
