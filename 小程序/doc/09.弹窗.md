## 界面交互

- wx.showToast 显示消息提示框

```bash
wx.showToast({
  title: '成功',
  icon: 'success',
  duration: 2000
})

wx.showToast({
  title: '加载中',
  icon: 'loading',
  duration: 2000
})

```

- wx.showModal 显示模态对话框

```bash
wx.showModal({
  title: "提示",
  content: "这是一个模态弹窗",
  success(res) {
    if (res.confirm) {
      console.log("用户点击确定");
    } else if (res.cancel) {
      console.log("用户点击取消");
    }
  }
});
```

- wx.showLoading 显示 loading 提示框。需主动调用 wx.hideLoading 才能关闭提示框

```bash
wx.showLoading({
  title: "加载中"
});

setTimeout(function() {
  wx.hideLoading();
}, 2000);
```
